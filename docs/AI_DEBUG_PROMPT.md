# SharpEmu AI Debug Prompt

Copy this prompt and paste it into ChatGPT, Claude, Gemini, or any AI assistant along with the debug report files.

---

## Prompt for AI Assistants

```
You are debugging SharpEmu, a PS5/PS4 emulator written in C#.

I will provide you with debug report files generated by SharpEmu's diagnostics system.

### Files provided:
- debug_report.json — structured crash report with root cause analysis
- missing-nids.json — database of all unresolved NID (function) imports
- crash-patterns.json — pattern matching rules for crash classification

### Your tasks:
1. Analyze the crash report to find the root cause
2. Identify which HLE export (NID) is missing or returning wrong values
3. Suggest which source file needs modification
4. Propose a code fix (C# snippet)
5. Compare with the game database to see if similar games had the same issue
6. Rate your confidence (0-100%)

### Rules:
- Do NOT suggest changes to unrelated emulator code
- Focus on the specific crash, not general improvements
- If the crash is in game code (not HLE), identify which HLE call should have prevented it
- If a NID is missing, look it up in the missing-nids database for context
- Prefer delegating to existing implementations (e.g., std::call_once → pthread_once)

### Output format:
1. **Root Cause**: [one sentence]
2. **Confidence**: [0-100%]
3. **Missing HLE**: [NID + function name, or "none"]
4. **Suggested Fix**: [C# code snippet or description]
5. **File to modify**: [path to source file]
6. **Similar games**: [list from game database]
```

---

## How to use

1. Run SharpEmu with debugging enabled:
   ```bash
   SHARPEMU_HEADLESS=1 SharpEmu game.bin
   ```

2. After crash, find the debug report:
   ```
   /home/z/my-project/download/crashes/crash_snapshot.txt
   data/ai-report-<gameid>.json
   ```

3. Copy the above prompt into your AI assistant

4. Attach the debug report files

5. The AI will analyze and suggest fixes

---

## Privacy

- No game files (eboot.bin, textures, shaders) are included in reports
- Only metadata, NIDs, register values, and crash diagnostics
- You control what you share with the AI
